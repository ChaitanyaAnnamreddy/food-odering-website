◉ CORS Extension For Fetching Swiggy Live API Data from Swiggy Public APIs. 
◉ Tailwind CSS 
◉ Class Based Components. 
◉ React Router DOM for routing & navigation 
◉ Lazy Loading 
◉ Context API 
◉ Lifting The State Up
